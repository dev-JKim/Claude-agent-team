# 세무/회계/노무 전문가 에이전트 팀

Claude Code Agent Teams를 활용한 세무·회계·노무 전문가 팀입니다.
질문을 입력하면 해당 분야 전문가 팀이 동시에 검토하고, 실무·법해석·리스크 관점에서 종합 답변을 제공합니다.

## 팀 구성 (12명 + 리드 1명)

```
🏢 총괄 매니저 (Opus)
│
├── 💰 세무팀
│   ├── 신고실무: 신고서식, 기한, 절차, 코드
│   ├── 세법해석: 법 조문, 예규, 판례 근거
│   └── 리스크검증: 세무조사 위험, 가산세
│
├── 📊 회계팀
│   ├── 재무회계: 분개, 계정과목, 재무제표
│   ├── 관리회계: 원가, 예산, 손익분석
│   └── 결산검증: 오류검증, 감사대비
│
├── 👷 노무팀
│   ├── 급여실무: 급여계산, 4대보험, 퇴직금
│   ├── 노동법해석: 근로기준법, 행정해석
│   └── 리스크검증: 노동청 진정 위험, 과태료
│
└── 🚢 관세/무역팀
    ├── 통관실무: 수출입 통관, 유니패스, HS코드, 관세계산
    ├── 관세법해석: 관세법, 대외무역법, FTA 특례법
    └── 리스크검증: 관세조사, 품목분류, 원산지 검증
```

## 참조 파일

- `references/기준수치-참조표.md`: 4대보험 요율, 최저임금, 비과세 한도 등 (매년 업데이트)
- `references/웹검색-규칙.md`: 공식 출처 사이트, 검색 금지 사이트 목록

## 사용 방법

1. Claude Code에서 이 저장소를 클론
2. 프로젝트 폴더에서 `claude` 실행
3. "에이전트 팀을 만들어줘" 입력
4. 질문하면 해당 분야 팀원이 동시 검토

## 답변 형식

최종 답변 시 선택 가능:
1. 간단 요약만
2. 요약 + 관련 법 조문 번호 포함
3. 요약 + 법 조문 + 실무 서식/양식 안내

## 주의사항

- 법률 자문이 아닌 정보 제공 목적입니다
- 중요한 의사결정은 반드시 전문가 상담을 받으세요
- AI의 답변은 참고용이며, 최신 법령과 다를 수 있습니다
