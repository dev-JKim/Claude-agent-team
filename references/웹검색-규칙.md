# 웹검색 연동 규칙

에이전트가 답변 시 웹검색을 활용하는 규칙이다.

---

## 반드시 검색해야 하는 경우

1. **최신 요율/수치 질문**: 4대보험 요율, 최저임금, 비과세 한도 등 매년 바뀌는 수치
2. **특정 예규/판례 확인**: 번호를 알고 있어도 내용을 검색으로 재확인
3. **법령 개정 여부**: "현재도 이 법이 유효한가?" 류의 질문
4. **신고 기한/서식**: 변경될 수 있으므로 최신 정보 확인

## 검색 금지 사이트 (블랙리스트)

```
❌ 네이버 블로그, 카페, 지식인
❌ 다음 카페, 브런치
❌ 개인 유튜브, 틱톡
❌ 세무사/회계사/노무사 개인 블로그
❌ 위키피디아, 나무위키
❌ 커뮤니티 (클리앙, 디씨, 레딧 등)
```

## 검색 결과 활용 규칙

1. **공식 사이트 결과만 인용**: 검색 결과 중 위 화이트리스트 사이트의 내용만 사용
2. **원문 확인**: 검색 결과 스니펫만 보지 말고, 해당 페이지를 직접 방문하여 원문 확인
3. **날짜 확인**: 검색된 자료의 작성일/시행일을 확인하여 현행 여부 판단
4. **출처 명시**: 답변에 출처를 명시 (예: "국세법령정보시스템 예규 서면-2023-법인-1234")
5. **검색 실패 시**: "해당 자료를 공식 사이트에서 확인하지 못했습니다. 직접 [사이트명]에서 확인해주세요."

## 검색 쿼리 작성 팁

```
좋은 예:
- site:taxlaw.nts.go.kr 상품권 근로소득 기타소득
- site:law.go.kr 소득세법 제20조
- site:glaw.scourt.go.kr 퇴직금 평균임금 판례

나쁜 예:
- 상품권 세금 처리 방법 (→ 블로그가 상위 노출됨)
- 퇴직금 계산 (→ 비공식 사이트 결과가 섞임)
```
